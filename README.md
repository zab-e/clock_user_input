Mini Time Checker

Description:
A simple C program that lets the user input a time in hours, minutes, and seconds, validates the input, and displays:

Total seconds

Total minutes (decimal)

12-hour format with AM/PM

24-hour format

Features:

Input validation for hours (0–23), minutes (0–59), and seconds (0–59)

Modular functions for calculations and formatting
